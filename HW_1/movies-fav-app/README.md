# Movies Favorite App — React + TypeScript + Vite

**ФИО:** _<Давлетов Руслан Раилевич>_  
**Группа:** _<БПИ235>_

Небольшое веб-приложение: показывает список фильмов, позволяет добавлять/удалять из избранного, фильтровать «Все/Избранные», искать по названию (без учёта регистра) и переключать режим отображения (плитка/список).

## Стек
- React 18, TypeScript 5, Vite 5
- Без сторонних UI-библиотек, стили — собственные

## Требования
- Node.js **≥ 18** (рекомендуется 20 LTS)
- npm (или yarn/pnpm)

## Быстрый старт
```bash
# установить зависимости
npm i

# создать файл окружения (пример ниже)
cp .env.example .env 2>/dev/null || true

# запустить dev-сервер
npm run dev
# открой адрес из консоли, обычно http://localhost:5173
